# PH GPS Analysis

Code for: Newman J, Ghani H, Munagala S, *et al.* The Pulmonary Hypertension Global Patient Survey: understanding the experiences and perspectives of patients. *ERJ Open Res* 2025; 11: 00297-2025. [DOI: 10.1183/23120541.00297-2025](https://doi.org/10.1183/23120541.00297-2025)

## Setup

Requires Python >= 3.11. Install dependencies with [Poetry](https://python-poetry.org/):

```
poetry install
```

Analysis is in `PVRI_analysis_final_cleaned.ipynb`.

## Data

Survey data is not included in this repository. Access can be requested from the Pulmonary Vascular Research Institute (PVRI): <https://pvrinstitute.org/learning-and-research/pulmonary-hypertension-global-patient-survey-phgps>
